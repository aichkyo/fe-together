# 滑动翻页组件

## 预览

![本地路径](./slide-pager.png)

## 使用指南

```javascript
import HrSlidePager from 'packages/hr-slide-pager/index';

export default {
  components: [HrSlidePager],
};
```

## 代码演示

### 基础用法

```vue
<HrSlidePager
  :currentId="id"
  :ids="ids"
  :canTurnPageFn="canTurnPageFn"
  :disabled="false"
  container="tic-detail-wrap"
  @turnPage="turnPage"
/>
```
